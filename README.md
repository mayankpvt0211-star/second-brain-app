### 🧠 Second Brain App

A clean, completely private, frontend-only digital sanctuary for your thoughts, notes, and AI-powered insights. 

I built this because I wanted a frictionless place to offload ideas, store memories, and dump text without worrying about where my data was going or who was tracking it.

This includes a web view, in which you can see how the tags assinged to the memories are realted to other memories, creating a more brain like environment as every memory can have multiple emotions and understanding towards it
## 🚀 How It Works

*   **100% Private Storage:** Your memories and notes are stored strictly on your local machine using your browser's `localStorage`. No databases, no external servers, and absolutely zero tracking. What you write stays on your device.
*   **Bring Your Own API Key:** To keep the deployment entirely secure and free from hardcoded credential leaks, the app uses a dynamic "Add API" input box. You just paste your own Groq or Gemini API key right in the UI, and it runs straight from your browser's temporary memory.
*   **Zero Backend Footprint:** Built as a pure HTML/JS frontend application, allowing it to be deployed instantly on Vercel with lightning-fast load times.

## 🛠️ Built With

*   Pure HTML5 & Modern JavaScript (ES6+)
*   Local Storage API (for secure, client-side data persistence)
*   Groq (for intelligent context retrieval)
*   Hosted beautifully on Vercel
